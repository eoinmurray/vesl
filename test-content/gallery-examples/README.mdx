# Gallery Component

The `<Gallery />` component provides a flexible way to display collections of images with a carousel interface and lightbox viewer.

## Basic Gallery

The simplest usage — just point to a folder containing images:

<Gallery path="./images" />

## With Title and Caption

Add context with `title`, `subtitle`, and `caption` props:

<Gallery
  path="./images"
  title="Research Results"
  subtitle="Phase 1 Experiments"
  caption="Experimental results showing signal propagation across different test conditions."
  captionLabel="Figure 1"
/>

## Limited Images

Use `limit` to show only a subset of images:

**Single image:**

<Gallery
  path="./images"
  limit={1}
  caption="limit={1}"
/>

**Two images:**

<Gallery
  path="./images"
  limit={2}
  caption="limit={2}"
/>

**Three images:**

<Gallery
  path="./images"
  limit={3}
  caption="limit={3}"
/>

## Pagination

Combine `limit` and `page` for paginated galleries:

**Page 1:**

<Gallery
  path="./images"
  limit={3}
  page={0}
  caption="Images 1-3"
/>

**Page 2:**

<Gallery
  path="./images"
  limit={3}
  page={1}
  caption="Images 4-6"
/>

## Using Glob Patterns

Target specific files with glob patterns:

<Gallery
  globs={["gallery-examples/images/image[1-3].png"]}
  title="Selected Images"
  caption="Using glob pattern to select specific images."
  captionLabel="Figure 3"
/>

## Cross-Directory Galleries

Pull images from multiple locations:

<Gallery
  globs={[
    "gallery-examples/images/image1.png",
    "example-imports/images/image5.png",
    "example-post/images/image9.png"
  ]}
  title="Mixed Sources"
  caption="Images collected from different directories in the content folder."
  captionLabel="Figure 4"
/>

## Props Reference

| Prop | Type | Description |
|------|------|-------------|
| `path` | `string` | Relative path to images folder |
| `globs` | `string[]` | Array of glob patterns |
| `title` | `string` | Gallery title |
| `subtitle` | `string` | Subtitle below title |
| `caption` | `string` | Caption text below gallery |
| `captionLabel` | `string` | Label prefix (e.g., "Figure 1") |
| `limit` | `number` | Max images to display |
| `page` | `number` | Page offset for pagination |

## Features

- **Carousel navigation** — Arrow buttons for browsing
- **Lightbox viewer** — Click any image to view full-size
- **Keyboard navigation** — Arrow keys and Escape in lightbox
- **Responsive** — Adapts from 1 to 3 columns based on viewport
- **Loading states** — Shimmer animation while images load
